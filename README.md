## Overview

This project is a web-based Learning Management System (LMS) developed during my software development internship. It provides an efficient platform for managing courses, students, instructors, and academic activities.

---

## Features

- User Authentication
- Student Management
- Course Management
- Admin Dashboard
- Responsive User Interface
- Database Integration
- Secure Login System

---

- PHP
- MySQL
- HTML5
- CSS3

---

## Installation

1. Clone the repository.
2. Copy the project folder into the `htdocs` directory (XAMPP).
3. Import the SQL database into phpMyAdmin.
4. Update database credentials if required.
5. Start Apache and MySQL.
6. Open the project in your browser.
